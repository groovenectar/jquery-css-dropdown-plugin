Features

  • Clean, minimalist design, ready to be styled. Ease of use and styling are the goals.
  • Supports variable-width menu items
  • Timers on open/close of submenus to reduce accidental menu operations
  • Option to automatically add arrows indicating sub menus
  • Option to automatically apply a drop shadow to sub menus
  • Customizable sub menu display animation using jQuery's animate() effects
  • Option for either horizontal or vertical menu
  • Functioning in tests with Chrome, Firefox, Safari, and Internet Exporer >= 7

Usage

Include the jQuery library, dropdown-menu.js and dropdown-menu.css on the page:

<link rel="stylesheet" href="dropdown-menu.css" />
<script type="text/javascript" src="lib/jquery-1.8.3.min.js"></script>
<script type="text/javascript" src="dropdown-menu.js"></script>

Use a standard UL/LI menu with anchors:

<ul class="dropdown-menu">
    <li><a href="#">Item 1</a></li>
    <li>
        <a href="#">Item 2</a>
        <ul>
            <li><a href="#">Submenu Item</a></li>
        </ul>
    </li>
</ul>

Apply the dropdown_menu() function in a Document Ready section:

<script type="text/javascript">
$(function() {
    $('.dropdown-menu').dropdown_menu();
});
</script>

See Example 1 and dropdown-menu-skin.css for styling hints.

See Example 2 source code for complete configuration options.

Example 2 - All Options Specified (Default Values)

Toggle Source

<script type="text/javascript">
$(function() {
    $('#example2 .dropdown-menu').dropdown_menu({
        sub_indicator_class  : 'dropdown-menu-sub-indicator',   // Class given to LI's with submenus
        vertical_class       : 'dropdown-menu-vertical',        // Class for a vertical menu
        shadow_class         : 'dropdown-menu-shadow',          // Class for drop shadow on submenus
        hover_class          : 'dropdown-menu-hover',           // Class applied to hovered LI's
        open_delay           : 200,                             // Delay on menu open
        close_delay          : 300,                             // Delay on menu close
        animation            : { opacity : 'show' },            // Animation for menu open
        speed                : 'fast',                          // Animation speed for menu open
        sub_indicators       : false,                           // Whether to show arrows for submenus
        drop_shadows         : false,                           // Whether to apply drop shadow class to submenus
        vertical             : false                            // Whether the root menu is vertically aligned
    });
</script>

Example 4 - Vertical Style Menu With Shadows and Arrows

For vertical menus, specifying the root menu's width is recommended, as well as setting its float value to none.

Toggle Source

<script type="text/javascript">
$(function() {
    $('#example4 .dropdown-menu').dropdown_menu({
        sub_indicators : true,
        vertical : true
    });
});
</script>

<style type="text/css">
#example4 .dropdown-menu {
    width: 260px;
    float: none;
}
</style>
